🡻 Description

Making games in Unity is not the only thing I do. I have actually made a game in Unreal Engine, Godot, and Java!

But besides games I have made a tool for ARK servers because I used to run a cluster with a friend. We had 5 servers in a cluster setup. So sending commands on all the servers at the same time was just a big inconvenience. Thats why I made a Discord bot using python.

It would send status updates about all the servers and you could send commands to the server like stopping or starting the servers, dino wiping or broadcasts.
